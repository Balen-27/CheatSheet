CSS Selectors & Specifity Cheat Sheet

Selectors

Selector Name Description
element Element selector Selects the elements from HTML document to style such as 'h2', 'p', 'body' etc.
.class Class selector This selects a class that has been defined within the HTML document and the rules will apply to all descendants too. This selector can be used as many times as needed throughout the HTML document.
#ID ID selector This will select the ID that has been defined within the HTML document and apply the rules to the element with that specific ID. An ID can only be used once in a HTML document unlike a class.

Specifity

Selector Specifity
#ID The most specific selector and over-rides all others
.class Less specific than the ID selector but over-rides basic element selectors.
element The least specific and selector. Is only applied if the previous selectors don't have rules that over ride the rules within the elemnt selector.
